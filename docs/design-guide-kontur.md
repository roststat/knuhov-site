# Дизайн-гайд: стиль Kontur (референс kontur.ru)

Извлечено с живого сайта kontur.ru. Это исходник для редизайн-клона в
`redesign-kontur/`. Оригинальная ЧБ-система Кнухова описана в `design-guide.md` —
этот файл её **не заменяет**, а фиксирует альтернативную визуальную систему для сравнения.

## ДНК стиля
Строгий монохромный каркас (near-black / white / серый) **+ сочные цветные акцентные плитки**.
Плотные жирные заголовки, bento-раскладка, кнопки-пилюли. Тёмный hero, светлые секции.

## Шрифт
- Оригинал: **Lab Grotesque** (проприетарный, без свободной кириллицы).
- Замена в клоне: **Golos Text** (Google Fonts, отличная кириллица, геометрический гротеск).
  Альтернатива — Inter Tight / Suisse Intl.
- Начертания: 400 / 500 / 600 / 700. Заголовки — 700, плотный `letter-spacing:-.02em`.

## Палитра
| Роль | HEX |
|---|---|
| Text near-black | `#222222` (текст), `#151515` (кнопки/тёмные секции) |
| Muted | `#858585`, `rgba(0,0,0,.48)` |
| Surface (светлый фон) | `#f4f5f6` |
| Accent синий | `#2563eb` / светлый `#2291ff` / тёмный `#015ccb` |
| Accent зелёный | `#1f9d55` / бирюзовый `#00bea2` |
| Accent янтарь | `#e0a526` |
| Accent красный (бренд Кнухова) | `#d02026` |

> В клоне цветные плитки взяты из бренд-палитры Кнухова, чтобы стиль Контура остался, а бренд — свой.

## Типографика (реальные значения с сайта)
| Уровень | Размер | Вес | Line-height |
|---|---|---|---|
| H1 (hero) | 56–64px | 700 | 1.0 (очень плотно) |
| H2 (секция) | 40–44px | 700 | 1.1 |
| Заголовок карточки | 24px | 700 | — |
| Подзаголовок | 18px | 500 | 1.4 |
| Body | 18px | 400 | 1.5 |

## Радиусы
- Карточки / плитки: **12px**
- Крупные блоки (форма, фото): 20–24px
- Кнопки и теги: **100px (пилюля)**
- Поля ввода: почти острые (2px у оригинала; в клоне 10px для мягкости)

## Компоненты
- **Bento-сетка** — плитки `repeat(3,1fr)`, gap 14px; часть широкие (`span 2`).
  Часть плиток — прозрачные тёмные, часть — цветные градиентные акценты.
  Разделители тонкие: `rgba(255,255,255,.14)`.
- **Кнопка primary** — `#151515` фон, белый текст, пилюля, padding ~15×26.
- **Кнопка ghost** — прозрачная + обводка, инверсия на hover.
- **Теги-пилюли** — мелкие чипсы поверх цветных плиток.
- **Список-карточки** на светлом `#f4f5f6`, hover — лёгкий подъём `translateY(-4px)`.
- **Форма-заявка** — тёмная карточка с закруглением, белые поля, focus-border синий.
- Контейнер **≈1128px**, центрирован, боковой padding 24px.
- Иконка-стрелка «на северо-восток» (↗) как знак «перейти».

## Файлы клона
- `redesign-kontur/index.html` — главная в стиле Контура (тот же контент, что `/index.html`).
- `redesign-kontur/kontur.css` — самодостаточная дизайн-система (не зависит от `/styles.css`).
- Внизу клона — плашка сравнения со ссылкой на текущий сайт. Клон помечен `noindex`.

## Откат
Клон изолирован в папке `redesign-kontur/`. Откат = удалить папку. Ничего в основном
сайте не тронуто.
