# S03 — Favicon-набор из мозаики бренда

## 1. Цель
Дать сайту фавикон (вкладка браузера, выдача Яндекса, иконка на домашнем экране) на основе фирменной мозаики квадратов — единственного графического актива бренда.

## 2. Контекст-бутстрап
Проект: knuhov.ru — статический сайт «Кнухов консалтинг» (медицинский консалтинг Ислама Кнухова). Плоский HTML/CSS/JS. Страницы собираются `node build.js` из `src/pages/` + `src/partials/`; общий `<head>` печатает шаблон `HEAD` в `build.js` (~строка 126) — сквозные теги добавлять туда. Корневые `*.html` — артефакты сборки. Хостинг — Vercel.
Прочитай первыми: `docs/plan/README.md`, `docs/plan/CONVENTIONS.md`, `docs/design-guide.md` (§11 — устройство мозаики: 3 квадрата в сетке 2×2, у каждого скруглён свой угол), `build.js`.
Глоссарий: «пуш» (команда владельца) = git add+commit → `git push origin main` → `vercel --prod --yes`. Без команды «пуш» не деплоить.

## 3. Предусловия/зависимости
Нет. Совместим с любым состоянием плана.

## 4. Затрагиваемые файлы
- `favicon.svg`, `favicon.ico`, `apple-touch-icon.png` (создать в корне; Vercel раздаёт корень как есть)
- `build.js` (шаблон `HEAD` — добавить `<link>`)

## 5. Точные инструкции
1. Создай `favicon.svg` — мозаика по design-guide §11: viewBox 0 0 64 64, фон прозрачный, три чёрных (#000) блока сетки 2×2 с gap ≈ 6: левый-верхний квадрат со скруглением только левого-верхнего угла (r=10), левый-нижний — только левого-нижнего, правая вертикаль на обе строки — скругление правых углов. Добавь `<style>@media (prefers-color-scheme:dark){rect{fill:#fff}}</style>` для тёмных вкладок.
2. Сгенерируй растровые версии (ImageMagick установлен на macOS через `magick`, либо `rsvg-convert`, либо `sips` после экспорта PNG):
   - `apple-touch-icon.png` 180×180 — БЕЛЫЙ фон (не прозрачный), мозаика с полями ~20%;
   - `favicon.ico` 32×32 (из того же PNG: `magick favicon-32.png favicon.ico`).
3. В `build.js`, в шаблон `HEAD` после `<link rel="stylesheet" ...>` добавь:
   ```html
   <link rel="icon" href="/favicon.svg" type="image/svg+xml">
   <link rel="icon" href="/favicon.ico" sizes="32x32">
   <link rel="apple-touch-icon" href="/apple-touch-icon.png">
   ```
4. `node build.js`.

## 6. Definition of Done
- [ ] Три файла иконок в корне; SVG — валидная мозаика в духе логотипа (не залитый квадрат).
- [ ] Теги `<link rel="icon">` присутствуют в `<head>` каждой собранной страницы.
- [ ] Фавикон виден во вкладке при локальном просмотре, в т.ч. в тёмной теме браузера.

## 7. Как проверить локально
`node build.js`; `npx serve .` → открыть `http://localhost:3000` — иконка во вкладке; открыть `/favicon.svg` напрямую и сравнить с мозаикой логотипа в шапке сайта.

## 8. Деплой
По умолчанию НЕ деплоить: изменения уходят пачкой по команде владельца «пуш». Если скомандовал: `git push origin main && vercel --prod --yes`.
