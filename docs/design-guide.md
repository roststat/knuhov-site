# Дизайн-гайд — Кнухов консалтинг

Основан на референсе strangehelix (значения сняты с живого сайта, точные). Система: строгий чёрно-белый, один шрифт без серифа, сквозной приём «один скруглённый угол». Контент и структура сайта — из ТЗ; здесь только визуальный язык.

---

## 1. Фирменный приём (главный токен)

**Один скруглённый угол — нижний правый, 24px; остальные три угла строго 0px.** Это не деталь кнопки, а сквозное правило: применяется ко всему прямоугольному с радиусом — меню, кнопки, карточки, плашки.

- `border-radius: 0 0 24px 0;` (TL 0 · TR 0 · BR 24px · BL 0)
- Полная граница `1px solid #000000` на всех сторонах (рамка полная, асимметричен только радиус).
- Логотип — мозаика из квадратов, у каждого свой угол скруглён по-разному; тот же приём повторяется в меню, кнопках и текстуре подвала. Это носитель узнаваемости.

---

## 2. Цвет

Строго ч/б, без акцентного цвета.

| Токен | Значение |
|---|---|
| Текст на светлом | `#000000` |
| Текст на тёмном | `#FFFFFF` |
| Фон страницы | `#FFFFFF` |
| Фон подвала | `#000000` (настоящий чёрный, не тёмно-серый) |
| Тени | нет вообще (`box-shadow: none`) |
| Градиенты | нет |

> Цветовой акцент гайдлайном не предусмотрен. Если позже захотим «сигнал потери» — это будет осознанное расширение, а не часть системы.

---

## 3. Шрифт

**Один шрифт везде.** Эталон референса — Roobert (Displaay, платный), геометричный гротеск среднего веса. Для бренда Кнухов выбран **Space Grotesk** (бесплатный, Google Fonts): геометричный, угловатый, средний вес (400–500) — заголовки тоже на ~500, не на жирном. На реальном сайте ближе всего к Roobert лягут General Sans или Switzer (Fontshare) — их и поставим в Claude Code; Space Grotesk — ближайшая замена, грузящаяся в превью. Никакого смешения sans + serif — только один гротеск, меняются вес и кегль (вес держим средний, без 800–900).

- Навигация и кнопки: **18px, weight 500, letter-spacing −0.18px**, цвет `#000`.
- Заголовки — тот же шрифт, просто очень крупно.

---

## 4. Меню (header)

- `position: fixed; top: 0; z-index: 100;` — плавает, не уезжает при скролле, не сжимается.
- Ширина 100% (полноширинная панель, не «остров» по центру).
- Высота **~80px**, padding `0 24px`.
- Фон `#fff`, граница `1px solid #000`, без тени.
- Углы: только нижний-правый 24px, остальные 0.
- Пункты навигации 18px / 500, чёрные.

---

## 5. Кнопки

Наследуют логику углов меню — системное решение.

- Углы: только нижний-правый 24px, остальные 0.
- Граница `1px solid #000`, padding ~24px, текст 18px / 500.
- **Secondary:** белый фон, чёрный текст («Пройти диагностику»).
- **Dark:** чёрный фон, белый текст («Заказать аудит»).
- Без теней, без других скруглений.

---

## 6. Подвал (footer)

- Фон `#000000`.
- **Верхняя граница — текстура из ряда квадратов с разными скруглёнными углами** (та же мозаика, что в логотипе, масштабированная на ширину). Структурный паттерн бренда, не картинка.
- Контент слева → справа: белый логотип + 2 предложения о бренде + строка благодарности команде; крупная ссылка-подпись (два слова, на отдельных строках) с **толстым подчёркиванием** (жирная горизонтальная черта, не `text-decoration`); колонка навигации (повтор меню); иконки соцсетей; кнопка «Help & Support» — фирменная капсула в белой обводке на чёрном.
- Нижняя строка (Политика / Наверх / Условия) — мелкий текст, разделители — **крошечные квадраты**, не круглые точки.

---

## 7. Дополнительные приёмы

- Толстое подчёркивание у крупных ссылок-подписей.
- Квадратные точки-разделители (маленький квадрат).
- Никаких теней и градиентов нигде.

## 9. Голос (из ТЗ — не меняется)

Разделение «мы» (компания) / «я» (Ислам); sentence case; слово «агентство» не используем — только «консалтинг».

---

## 10. Привести в соответствие

- **Логотип:** текущая версия скруглена по всем углам — переделать под фирменный приём (мозаика квадратов или один угол на знаке).
- **Маршрутизатор и лента слоёв:** убрать цветовую подсветку — состояния в ч/б через чёрную заливку/инверсию и угловой приём.
