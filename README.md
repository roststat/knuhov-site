# Кнухов консалтинг — сайт

Статический многостраничный сайт. Главная — воронка одного входа в стратегический аудит; продукты — отдельными лендингами. Дизайн-система: строгий ч/б, фирменный приём «один скруглённый угол» (нижний-правый 24px), один гротеск.

## Структура

| Файл | Назначение |
|---|---|
| `index.html` | Главная: герой → маршрутизатор заблуждений → где теряются деньги → метод → дорожная карта → сопровождение → кейсы → бутиковый формат → манифест → аудит (50 000 ₽) + форма |
| `produkty.html` | Витрина продуктов |
| `avtozabota.html` | Лендинг «Автозабота» (удержание и возврат пациентов) |
| `ii-administrator.html` | Лендинг «ИИ-администратор» |
| `kontent-zavod.html` | Лендинг «Контент-завод» |
| `vidzhety.html` | Лендинг «Виджеты» |
| `logo.png` / `logo-white.png` | Логотип и его белая версия (для тёмного подвала) |
| `docs/TZ.md` | Полное ТЗ |
| `docs/design-guide.md` | Дизайн-гайд (токены) |

Открывается локально двойным кликом по `index.html` — всё самодостаточно (стили и логотип встроены в страницы).

## Дизайн-система (кратко)

- Цвет: `#000` на белом, `#fff` на чёрном подвале. Без теней и градиентов. (Цвет есть только в логотипе.)
- Фирменный угол: `border-radius: 0 0 24px 0` на меню, кнопках, карточках.
- Меню: фиксированное, высота 104px (логотип увеличен 2x).
- Шрифт: сейчас Space Grotesk (Google Fonts).

## Рекомендации при сборке (для Claude Code)

- CSS сейчас инлайнен в каждую страницу (ради самодостаточного превью). При желании вынести в общий `styles.css` и подключить ссылкой.
- Логотип встроен как data-URI. Можно перевести `<img>` на внешние `logo.png` / `logo-white.png`.
